## 已解决事项

### 1. 题库查询接口

状态：已解决

接口：

```http
GET /api/v1/question-items
```

已支持：

- 按 `generation_batch_id` 查询全批次题目。
- 按 `paper_result_id`、`knowledge_point_id`、`question_type`、`difficulty_level`、`scene_type` 筛选。
- 返回分页结构，与现有列表接口保持一致。
- 返回题干、答案、解析、题型、难度、知识点、所属试卷和测练场景等前端展示字段。

### 2. 覆盖报告课件页码解析

状态：已解决

已完成：

- `slide_no` 为数字或字符串数字时正常解析。
- `slide_no` 缺失、空值或无法解析时回退到遍历序号。
- 异常课件页码不再导致覆盖报告刷新失败。

### 3. 覆盖报告课件统计口径

状态：已解决

已完成：

- 覆盖矩阵只纳入 `result_status=success` 的课件结果。
- 避免 pending、processing、failed 课件污染演示口径。

### 4. 后端文档旧描述同步

状态：已解决

已完成：

- 多场景测练不再描述为仅默认 `unit_test`。
- 试卷 DOCX 导出已写入说明。
- 覆盖报告纳入题目和课件页面已写入说明。
- `question_item` 可作为题库基础能力已写入说明。
